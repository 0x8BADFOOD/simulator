# LGSVL Simulator

New and improved Autonomous Vehicle Simulator!

## Building WebUI

1. Install recent version of [node.js](https://nodejs.org/en/) with npm, LTS 10.15 works.

2. Run `npm install` inside `WebUI` folder to install dependencies, do this only once
   or if dependencies change inside `packages.json` file.

3. Run `npm run pack-p` to create optimized html/js/css files.
